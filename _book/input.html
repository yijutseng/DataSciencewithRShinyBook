Ch. 3 輸入Inputs

fluidPage(
  #input()
  #,
  #output()
)

input函數的使用方法大同小異，包括inputId,label,...:

  • inputId: 為輸入元件的名稱，為每個元件取合理且唯一的名稱，才可在Shiny App程式中取用
  • label: 為輸入元件在畫面上呈現的名字，顯示用，因此必須輸入使用者看得懂的說明
  • …: 每個輸入元件會有自己的特殊設定參數，可用?輸入元件函數名稱來查詢說明文件
sliderInput(
  inputId = "num",
  label ="Choose a number",
  ...
)
